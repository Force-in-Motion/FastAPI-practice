# FastAPI Practice Project

Этот проект представляет собой практическое применение FastAPI для создания простого API с различными эндпоинтами.

## Описание

Проект содержит несколько эндпоинтов, каждый из которых выполняет определенную функцию:

1. Генерация пароля
2. Расчет силы притяжения на планетах
3. Расчет штрафа за превышение скорости
4. Конвертация валют

## Установка

1. Клонируйте репозиторий:
   git clone <url-вашего-репозитория>

2. Перейдите в директорию проекта:
Apply
   cd FastAPI-practice/hw_1

3. Установите зависимости:
Apply
   pip install fastapi uvicorn

## Запуск

Для запуска сервера выполните:
Apply
python main.py

Сервер запустится на http://localhost:8000.

## Эндпоинты

### 1. Генерация пароля
- URL: /password
- Метод: GET
- Параметры: length (длина пароля)
- Пример: http://localhost:8000/password?length=10

### 2. Расчет силы притяжения
- URL: /gravity
- Метод: GET
- Параметры: 
ame_planet (название планеты), height (высота над поверхностью)
- Пример: http://localhost:8000/gravity?name_planet=Earth&height=100

### 3. Расчет штрафа за превышение скорости
- URL: /speeding-fine
- Метод: GET
- Параметры: speed (текущая скорость), limit (ограничение скорости)
- Пример: http://localhost:8000/speeding-fine?speed=120&limit=90

### 4. Конвертация валют
- URL: /convert/{from_currency}/{to_currency}
- Метод: GET
- Параметры пути: rom_currency, 	o_currency
- Параметры запроса: mount (сумма), commission_rate (комиссия)
- Пример: http://localhost:8000/convert/USD/EUR?amount=100&commission_rate=0.01

## Разработка

Проект использует FastAPI и сервисный слой (Service класс) для выполнения бизнес-логики. Для добавления новых эндпоинтов, расширьте main.py и добавьте соответствующие методы в Service класс.
